# zhihu-daily-desktop

> zhihu daily, desktop-version

## Screenshots

![gif](/screenshots/zhihu-daily-desktop.gif)

## Features

- **Night Mode**
- **infinite scroll**

## Tech Stack

- **Vuejs 2.x**
- **Electron**
- **electron-vue**
- **Vuex**
- **vue-router**
- **axios**

## Goals

- [x] Routing
- [ ] 60fps animations(flip)
- [x] Night mode
- [ ] Unit test

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:9080
npm run dev

# build electron app for production
npm run build
```
